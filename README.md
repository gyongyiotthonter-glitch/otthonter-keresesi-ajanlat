# 🏠 Ingatlan Ajánlat Generátor

**Takács Adél Ingatlan** – Streamlit Cloud alkalmazás

## Felhasználás

1. Megnyitod az alkalmazás linkjét a böngészőben
2. Beilleszted az ingatlanbazar.hu hirdetések URL-jeit
3. Megadod az ügyfél nevét
4. PDF letöltés gomb → kész!

## Telepítés Streamlit Cloud-ra (ingyenes)

1. Hozz létre GitHub fiókot: https://github.com
2. Új repository létrehozása, fájlok feltöltése
3. Menj: https://streamlit.io/cloud → Sign in with GitHub
4. "New app" → válaszd ki a repository-t → Deploy

## Helyi futtatás

```bash
pip install -r requirements.txt
streamlit run app.py
```
